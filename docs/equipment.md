# Panduan Menu Equipment

Menu **Equipment** pada aplikasi ini digunakan untuk mengelola data seperti data Daily Report, P2H, Mechanical Inspection, Repair & Maintenance, Purchase Requisition. Berikut adalah penjelasan yang lebih sederhana mengenai cara mengelola setiap bagian di menu **Equipment**.

---

## 1. Daily Report

Halaman **Daily Report** digunakan untuk mengelola data laporan harian pengiriman LCT dalam sistem. Di halaman ini, Anda dapat melihat daftar **Daily Report** yang sudah tercatat, menambah laporan harian baru, mengedit laporan yang sudah ada, atau menghapus laporan yang tidak diperlukan.

Menu **Daily Report** berfungsi untuk menyimpan informasi terkait data muatan, penggunaan fuel, serta durasi pengiriman LCT. Data ini membantu pengguna dalam memantau aktivitas pengiriman, mengevaluasi konsumsi bahan bakar, dan mengetahui waktu yang dibutuhkan dalam setiap proses pengiriman.

### 1.1 Menambah Daily Report

Untuk menambahkan **Daily Report** baru, ikuti langkah-langkah berikut:

- Klik tombol **"+ New"** yang terletak di bagian atas kanan halaman. Akan muncul formulir untuk memasukkan data baru.

![Daily Report Index](/assets/images/daily_report/index.png){ style="display: block; margin: 0 auto;" }

- Isi semua informasi yang diperlukan, seperti data muatan, fuel, dan durasi pengiriman LCT.

![Daily Report Add](/assets/images/daily_report/add-button.png){ style="display: block; margin: 0 auto;" width="50%" }

- Setelah selesai, klik tombol **Save** untuk menyimpan **Daily Report** baru.

### 1.2 Mengedit Daily Report

Untuk mengedit **Daily Report** yang sudah ada, ikuti langkah-langkah berikut:

- Pada daftar daily report, temukan data yang ingin Anda edit. Klik tombol **Action** di sebelah data tersebut. Pilih opsi **Edit** dari dropdown yang muncul.

![Daily Reprot Index](/assets/images/daily_report/edit-button.png){ style="display: block; margin: 0 auto;" width="50%" }

- Ubah informasi yang ingin diperbarui di formulir edit.

![Daily Reprot Index](/assets/images/daily_report/save-button.png){ style="display: block; margin: 0 auto;" width="50%" }

- Setelah selesai, klik tombol **Save** untuk menyimpan perubahan.

### 1.3 Menghapus Daily Report

Untuk menghapus **Daily Reprot** yang tidak diperlukan lagi, lakukan langkah-langkah berikut:

- **Klik Tombol "Action"** di sebelah kanan **Daily Report** yang ingin Anda hapus. Tombol **Action** berada di samping setiap **Daily Report** yang terdaftar dalam tabel.

![Daily Reprot Index](/assets/images/daily_report/delete-button.png){ style="display: block; margin: 0 auto;" width="50%" }

- **Pilih "Delete"** dari dropdown menu. Setelah memilih **Delete**, sistem akan meminta konfirmasi untuk memastikan Anda ingin menghapus **Daily Report** tersebut.

- **Konfirmasi Penghapusan** untuk menghapus **Daily Report** dari daftar.
![Daily Reprot Index](/assets/images/daily_report/confirm-button.png){ style="display: block; margin: 0 auto;" width="50%" }

## 2. P2H

Menu **P2H** berfungsi untuk menyimpan informasi terkait kondisi kendaraan, hasil pemeriksaan komponen, kelayakan operasional, serta catatan temuan atau kerusakan pada kendaraan. Data ini membantu pengguna dalam memantau kesiapan kendaraan sebelum digunakan, memastikan kendaraan dalam kondisi aman, serta mendukung proses perawatan dan evaluasi operasional kendaraan.
